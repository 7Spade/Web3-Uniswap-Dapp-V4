# 系統模式記憶庫

## 架構模式
- **前端架構**: 組件化 React 應用
- **路由模式**: Next.js App Router
- **狀態管理**: 本地狀態 + Context API
- **樣式模式**: Utility-first CSS (Tailwind)

## 代碼組織模式
- **組件結構**: 功能導向的組件組織
- **文件命名**: kebab-case
- **導入模式**: 相對路徑導入
- **類型定義**: 集中式類型管理

## Web3 整合模式
- **錢包連接**: Wagmi hooks
- **交易處理**: Viem 客戶端
- **狀態同步**: 實時區塊鏈數據
- **錯誤處理**: 統一的錯誤邊界

## 開發模式
- **熱重載**: Next.js 開發服務器
- **類型檢查**: TypeScript 編譯時檢查
- **代碼格式化**: Prettier
- **代碼檢查**: ESLint

## 部署模式
- **構建工具**: Next.js 內建構建系統
- **靜態資源**: Next.js 優化
- **環境變數**: 運行時配置
